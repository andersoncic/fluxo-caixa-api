Camadas Principais:

Camada de Apresentação (Presentation Layer)

API Gateway (FastAPI)

Exposição de endpoints REST:
    /transactions: Para gerenciar lançamentos (créditos/débitos).
  • /summary: Para retornar o saldo diário consolidado.
  • Camada de Aplicação (Application Layer)

Serviço de Gerenciamento de Lançamentos

  • Valida e processa requisições de crédito/débito.
  • Insere dados na base de dados.
  • Serviço de Consolidação Diária
  • Calcula o saldo diário com base nos lançamentos armazenados.
  • Camada de Dados (Data Layer)

    Banco de Dados SQLite

  • Tabela transactions:
  • id: Identificador único.
  • date: Data do lançamento.
  • type: Tipo do lançamento (crédito ou débito).
  • amount: Valor da transação.
